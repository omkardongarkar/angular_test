Agent Performance

Top 5 Things Agents Did Right
  1. Quick Response Time: Responded to 90% of user queries within 5 minutes.
  2. Accurate Information: Provided correct product details in 95% of interactions.
  3. Successful Issue Resolution: Resolved return and refund requests in 85% of cases.
  4. Proactive Assistance: Suggested solutions before users explicitly asked, improving satisfaction by 20%.
  5. Consistency in Policy Explanation: Clear and consistent communication of return and warranty policies.
Top 5 things agent Did Wrong
  1. Refund Delays: Failed to expedite refund processes in 30% of cases, causing dissatisfaction.
  2. Incomplete Resolutions: Escalated 15% of simple issues due to lack of clarity on workflows.
  3. Miscommunication: Provided conflicting information on shipping times in 10% of interactions.
  4. Unempathetic Responses: Displayed a lack of empathy in handling user complaints about defective products.
  5. Ineffective Follow-ups: Failed to follow up on unresolved tickets in 25% of instances.
Top 5 Recommandations Moving Forward
  1. Improve Refund Process Efficiency: Streamline workflows to reduce refund processing delays.
  2. Enhanced Training: Provide training on consistent communication and conflict resolution.
  3. Proactive Issue Management: Equip agents to anticipate common user concerns and address them preemptively.
  4. Empathy Modules: Incorporate empathy training for handling sensitive user issues.
  5. Better Follow-Up Protocols: Implement reminders and automated follow-ups for unresolved tickets.
